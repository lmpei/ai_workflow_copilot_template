# Stage K-03: Latest Source Intake And Research Record Foundation

## Goal

Introduce one bounded workflow foundation for `AI 前沿研究` that starts from current high-trust source material and
stores durable research records instead of treating the module as generic workspace chat only.

## Scope

- define one bounded latest-source intake contract
- define one durable research-record contract for:
  - source set
  - derived themes
  - derived events
  - derived projects
  - summary
  - judgment
- connect the MCP-backed source path to this new intake shape rather than leaving MCP as an isolated protocol demo

## Out Of Scope

- full source crawling breadth
- multi-server MCP marketplace work
- final workbench redesign

## Deliverables

- backend contract for latest-source intake
- backend contract for one durable research record
- one bounded MCP-backed path that feeds the new record model

## Verification

- backend tests
- frontend type and integration verification if contracts change
