# AI 热点追踪简报阅读布局修复

## Objective

修复公网 `AI 热点追踪` 页面中简报阅读区过小、追问发送按钮不可见、追问回答区过小的问题。

## Scope

- 左侧简报改为整块面板内部滚动，不再把标题区和正文区割裂成导致正文很矮的两段。
- 缩短简报头部与运行态区域占用高度，让主信号、继续关注和盲点拥有更大的阅读面积。
- 右侧追问改为固定底部输入栏，发送按钮始终可见。
- 追问聊天记录区获得主要剩余高度，预设问题更紧凑。
- 保留可见细滚动条，避免无鼠标时难以发现内容可滚动。

## Verification

- `npm --prefix web run verify`
- 生产页面手工检查简报和追问区域。
